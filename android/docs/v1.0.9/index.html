    Mappls Map Android SDK

    Easy To Integrate Maps & Location APIs & SDKs For Android Applications

    Powered with India's most comprehensive and robust mapping functionalities.

    1. You can get your api key to be used in this document here: https://about.mappls.com/api/signup

    2. The sample code is provided to help you understand the basic functionality of Mappls maps & REST APIs working on Android native development platform.

    3. Explore through 238 nations with Global Search, Routing and Mapping APIs & SDKs by Mappls.
